I'm **Gabriel Balbine** — Computer Science undergrad (8th semester @ FEI) specializing in **AI, computer vision, and deep learning** applied to real-world problems.

## 🔬 Featured Project — GAIA / TITAN

> End-to-end computer vision pipeline for automated behavioral analysis in ASD therapy sessions, assisting clinicians with objective, quantitative metrics extracted from interaction videos.

```
Video → YOLOv8-Pose + BoTSORT → MediaPipe Face Mesh → Feature Extraction → LSTM Classifier
```

| Metric | Result |
|--------|--------|
| Session-level Accuracy | **83.3%** |
| AUC-ROC | **0.759** |
| Specificity | **82.9%** |
| Sensitivity | **51.9%** |
| F1-Score | **60.4%** |
| Validation | LOSO — 7 folds, participant-level |

**Key technical challenges solved:**
- Role disambiguation in similar-size subject pairs via adaptive lock thresholds and pair-based prescan
- Multi-view augmentation strategy (8 NT + 4 ASD sessions) to maximize training data without data leakage between cameras of the same participant
- Privacy-preserving face-only elliptical blur for ethical compliance
- Full Streamlit dashboard (GAIA App) with role-based auth, risk gauges, and automated JSON ingestion

🛠️ Python · PyTorch · YOLOv8 · BoTSORT · MediaPipe · Streamlit · Plotly
